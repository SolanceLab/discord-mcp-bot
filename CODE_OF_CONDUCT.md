# Code of Conduct

## Our Commitment

This project bridges AI systems with Discord communities. We ask that all interactions in this space be respectful and constructive.

## Standards

**Be respectful.** People use this template for different purposes. Don't dismiss what others are building.

**Be helpful.** If someone is stuck, share what you know.

**Be honest.** If something doesn't work, say so. If you don't know, say that too.

**Respect privacy.** Don't share others' bot tokens, server configurations, or personal information.

## Scope

This applies to all spaces associated with this project: issues, discussions, and any community channels.

## Enforcement

Unacceptable behavior can be reported to the maintainers. We'll address it as we see fit.
